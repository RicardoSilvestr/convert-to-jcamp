src/index.js

Convert strings into JCAMP and add extra information

index(data: string, options: object?): string
Parameters
data (string) values to add to the file, usually a csv or tsv values
options (object? = {})
Name Description
options.title string (default '') title of the file
options.owner string (default '') owner of the file
options.origin string (default '') origin of the file
options.type string (default '') type of data
options.xUnit string (default '') units for the x axis
options.yUnit string (default '') units for the y axis
options.info object (default {}) comments to add to the file
Returns
string: JCAMP of the input
